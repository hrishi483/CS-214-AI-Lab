How to run the program and get the output:
1.Save the zip file and extract into a folder in your computer
2.Open the python file in your python editor (I used PyCharm)
3.Keep the start in input1.txt file and goal in goal1.txt
4.Open terminal and write py -3 .\15.py input1.txt goal1.txt
5.output is generated in the console itself. 
6.For each heuristic function the cost and the time for execution is printed,and the output for
   each heuristic function is separated.
